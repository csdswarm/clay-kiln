
In order to get this POC working:

cd claydemo/spa 

yarn install

and run 

yarn build

then 

cd claydemo

Then copy the built vue app.js file into the claydemo app public folder like so

cp spa/dist/app.js app/public/dist/app.js

Then navigate to a page and make sure you are in edit mode

example

http://clay.radio.com/_pages/index.html?edit=true
